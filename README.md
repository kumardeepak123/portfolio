## Setup

##### Clone the repository

```bash
   git clone https://github.com/kumardeepak123/portfolio.git
```
##### Move to the desired folder
```bash
   cd \portfolio
```
##### To run the app 
```bash
  Click on the index.html file .It will automatically open the web page on your browser.
```
